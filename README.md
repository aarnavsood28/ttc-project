# CubeSat Communication System Resources

This repository contains resources developed as part of CubeSat communication system design and testing. The files are organized into three main components:

## Contents

### Turnstile Antenna Simulations
- ANSYS HFSS simulation files for a turnstile antenna integrated with a 1U CubeSat frame.
- Includes the complete simulation model used to analyze the antenna's radiation characteristics and performance.

### Link Budget Analysis
- MATLAB scripts for evaluating the communication link budget between the CubeSat and the ground station.
- A PDF report documenting the link budget calculations and methodology is included for reference.

### LoRa Communication Code
- Arduino code for interfacing with a LoRa transceiver.
- Implements packetization and depacketization of sample beacon data for communication testing.
